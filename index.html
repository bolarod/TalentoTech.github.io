Proyecto final Análisis y Visualización de Datos

Estudiante: Alex Bolaños Rodriguez

El proyecto se centra en analizar los datos de ventas por año registrados por las estaciones de servicio que dispensan gas natural a nivel nacional

Metodología

Para realizar este proyecto se toma como fuente el portal de datos abiertos del gobierno de Colombia (www.datos.gov.co),allí obtuvimos un set de datos correspondie a Ventas de Gas Natural Comprimido Vehicular. Para almacenar los datos creamos una base de datos MySQL en la nube clever-cloud para luego, por medio de código python, poblar la tabla y disponer de la información para generar un Dashboard en Looker Studio de Google. Finalmente, se crea una pagina HTML que muestra un Dashboard y Mapa de calor (en Javascript y Leaflet) embebidos para facilitar visualizar y análizar los datos de manera sencilla.

Mapa de Calor (Heatmap)

Mediante el uso de la librería Folium realizamos un mapa de calor para identificar las zonas del pais en donde mas se concentran las estaciones de servicio que dispensan gas Natural.

Conclusiones

La visualización de datos en la nube por medio de plataformas y lenguajes abiertos nos permiten generar soluciones de manera interconectada, lo que facilita su desarrollo y flexibiliza su desarrollo. En este curso aprendí a integrar bases de datos, lenguaje python/Javascript y plataformas de visualización como Lookerstudio para generar tableros de control y mapas dinamicos que sin lugar a dudas generan valor para las personas que toman decisiones.
